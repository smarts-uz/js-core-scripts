# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/), and the project follows
[Conventional Commits](https://www.conventionalcommits.org/) — so this file can
be regenerated from the git history.

## [Unreleased]

### Added

- **Per-method runner architecture** — every public static method of every
  `utils/` class is exposed as a standalone runner at `runs/<Class>/<method>.mjs`,
  generated by `runs/_generate.mjs`. Replaces the per-operation root scripts.
- Auto-generated `README.md` (`runs/_gen-readme.mjs`) — one section per class and
  method, with JSDoc, run command, and parameter tables.
- **Project quality tooling** — ESLint (flat config), Prettier (with
  format-on-save and a Husky + lint-staged pre-commit hook), `.editorconfig`,
  `.nvmrc`, and Jest coverage thresholds.
- **CI/CD** — GitHub Actions (`lint-and-format` on Ubuntu, `test` on Windows),
  Dependabot, PR/issue templates, and `CODEOWNERS`.
- Governance docs — `LICENSE` (MIT), `CONTRIBUTING.md`, this `CHANGELOG.md`.

### Changed

- Each runner is now a complete standalone script that calls
  `<Class>.<method>(...)` directly (no shared runtime engine).

### Security

- Credentials migrated from `config.yml` / source into a gitignored `.env`
  (read via `utils/Secrets.js`). Tokens previously committed to history must be
  rotated — see project notes.
